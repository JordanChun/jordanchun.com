Pißwasser - Website

A website redesign of a craft beer company. Users can browse through a section of craft beers or learn about the factory. Created with standard HTML, CSS, Javascript, jQuery, and PHP.

As the website was designed by a partner, it was my job to bring the design to life. The website was coded using HTML and CSS. The website was coded pixel perfect to match the website design.

The website was coded and designed with a fluid layout to allow resizing of the window and mobile usage without causing issues.

Information about the beer is retrieved from the database using Ajax.
